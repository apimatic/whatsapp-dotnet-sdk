
# Audio

## Structure

`Audio`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |

| `Link` | `string` | Optional | The protocol and URL of the media to be sent. Use only with HTTP/HTTPS URLs. Required when you are not using an uploaded media ID. |
